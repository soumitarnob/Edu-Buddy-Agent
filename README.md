## Project Overview
EduBuddy is a simulated AI study assistant built as part of the Kaggle 5-Day AI Agents Intensive capstone. It can summarize material, generate flashcards, produce quizzes, run simulated tools, and store session logs in memory.

## Features
- Memory system  
- Web search (simulated)  
- Flashcards  
- Quiz generator  
- Session logging  
- Demo workflow

## Run
```
python edubuddy_agent.py
```
